## Getting Started

1. Make sure you have [Node.js 20.10.0](https://nodejs.org/en/download)
2. Clone the repo
3. Copy `.env.example` to `.env`
4. Run:

```bash 
npm i
```

```bash
npm run dev #client
```

```bash
npm start #server
```
